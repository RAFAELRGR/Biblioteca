## 🚀 Instalación y ejecución

### Requisitos
- Node.js 18+
- npm

### Pasos

**1. Instalar dependencias**
```bash
npm i
```

**2. Build de producción**
```bash
npm run build
```

**3. Correr el proyecto**
```bash
npm start
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
